# 2K Scout

NBA 2K scouting tool that analyzes matchups between two teams and generates an AI-powered game plan based purely on 2K attribute ratings.

## What it does

Select any two teams (including All-Time rosters) and 2K Scout automatically computes:

- **Attack Offensively** — opponent's weakest defenders to ISO
- **Hide Defensively** — your weakest perimeter defenders paired with their worst shooters so they can sag off
- **Leave Open** — opponent perimeter players safe to leave unguarded
- **Speed Advantage** — your players with a significant speed edge over their positional matchup
- **Best Matchups** — your best available defender for each of their top 5 players (position-aware, stretch bigs handled correctly, no defender assigned twice)
- **My Team's Best Badges** — top players with their Legendary/HoF/Gold badges
- **AI Game Plan** — a full offensive/defensive scouting report generated by Claude, reasoning strictly from 2K stats

## Architecture

```
Browser (React SPA)
    │
    ├── /api/*  ──────────────────▶  nba2kapi.com  (player & roster data)
    │            Nginx proxy
    │
    └── Lambda URL  ──────────────▶  AWS Lambda (Node.js)
                                          │
                                          └──▶  Anthropic Claude API
                                               (AI game plan generation)

EC2 (Amazon Linux)
  └── Nginx
        ├── Serves static React build (dist/)
        └── Proxies /api/* to nba2kapi.com
```

## Stack

| Layer | Tech |
|---|---|
| Frontend | React + Vite |
| Styling | Custom dark-theme CSS |
| Static hosting | EC2 + Nginx |
| API proxy | Nginx (avoids browser CORS) |
| AI backend | AWS Lambda (Node.js ESM) |
| AI model | Anthropic Claude (`claude-opus-4-8`) |
| Player data | nba2kapi.com |

## Local development

```bash
cp .env.example .env
# Add your NBA 2K API key to .env

npm install
npm run dev
```

The Vite dev server proxies `/api/*` to `https://api.nba2kapi.com` automatically.

Set `LAMBDA_URL` in `src/App.jsx` to your deployed Lambda function URL to enable AI game plans locally.

## Deploying to EC2

```bash
cd /opt/2k-scout
sudo git pull
sudo npm ci
sudo npm run build
sudo systemctl restart nginx
```

## Lambda (AI game plan)

The Lambda function in `lambda/` proxies requests to the Anthropic API, keeping the API key server-side.

```bash
cd lambda
npm install
# Windows:
Compress-Archive -Path .\* -DestinationPath ..\lambda.zip
# macOS/Linux:
zip -r ../lambda.zip .
```

Upload `lambda.zip` to your AWS Lambda function. Set `ANTHROPIC_API_KEY` as a Lambda environment variable and increase the timeout to at least 60 seconds.
